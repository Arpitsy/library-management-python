# 📚 Library Management System (Python)

A simple Library Management System built using Python and Object-Oriented Programming (OOP) concepts.

## 🚀 Features

* Add books with categories
* Prevent duplicate book entries
* Display all books and total count
* Search and view book categories
* Remove books from the library

## 🧠 Concepts Used

* Classes and Objects
* `__init__` constructor
* Class variables vs Instance variables
* Lists / Dictionaries for data storage
* Conditional logic and control flow

## 💡 Learning Outcome

This project helped me understand how Python classes work in real scenarios.
I learned how `__init__` automatically runs when creating objects and how to manage shared data across multiple instances.

It also improved my understanding of:

* Writing cleaner logic
* Avoiding duplicate data
* Structuring small systems using OOP
